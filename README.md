# OctoSpray-v2
OctoSpray is a python brute forcing tool created specificly for hosts with Octoprint installed.


usage: OctoSpray.py [-h] [-v] (-l USER | -L USRLIST) (-P PWDLIST | -p PASSWORD) (-u URL | -U URLLIST)
